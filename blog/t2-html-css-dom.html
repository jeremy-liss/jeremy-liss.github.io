What are HTML CSS and the DOM?

The DOM is a bit like a tree. Each branch of the tree is created by an HTML element. CSS is used to change the style and appearance of an HTML element.

What is Boxifiying?

To boxifiy a design means to take a design brief and divide its elements up into boxes, the boxes being the HTML elements you will create with code.

What is the box model?

Each element of HTML sits within a box which can be styled with CSS and is known as the box model. The box is made up of margin, border, padding and content.
