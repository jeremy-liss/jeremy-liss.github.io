In javascript, scope refers to the current context of your code, which can be either global or local. If you declare a variable in the main part of your code, it has a global scope and can be referred to in any other part of the code. If you declare a variable with a function or a loop etc., then it has a local scope and can only be referred to within that function etc. A useful thing to remember is that a piece of code can call up any variables or functions from the code it is enclosed within, but not from a piece of code that it encloses.
