Parentheses ()

  1. Executing a function - var result = doSomething()
  2. Declaring something to print to the console - console.log("something")
  3. Containing an expression - if (3 < 5), for (var i = 0 etc.)
  4. Grouping expressions - var i = (3 +5) * 2

Brackets []

  • Square brackets are used just to contain arrays - var arr = [1, 5, 7]

Braces{}

  1. Around object bodies - var obj = {size: big}
  2. Around function bodies - function doSomething () {do something}
  3. Around conditional bodies - if (1 =1) {do something}
  4. Around loop bodies - for (var i = 0 etc.){do something}

Single and double quotes

  • Quotes are use to denote strings in javascript. You can use either double or single quotes, however using double quotes means you can use single quotes within the string for contractions or to include quotes within the string. Using single quotes would mean the string would end if you added a contraction or quote mark. - "He's saying 'good'"
